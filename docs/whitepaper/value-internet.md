# Value Internet (Public Overview)

Acee defines the Value Internet as a network where influence, identity, AI behaviors and RWA can be:
1) standardized, 2) verified, 3) transacted, and 4) coordinated via incentives/governance.

Layers:
- Asset Standard Layer
- Value Coordination Layer
- Network/Application Layer (ProjectEX)
- Value Layer (POI)
