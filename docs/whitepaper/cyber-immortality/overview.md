## Technical Architecture / 技术架构

### Smart Contracts / 智能合约

**AchievementBadges Contract:**
- ERC-721 based achievement system
- Configurable badge types
- Soulbound token (SBT) functionality
- Event tracking and verification

**ImmortalityBadge Contract:**
- Special recognition for long-term participants
- Unique badge with special privileges
- On-chain verification
- Cross-platform recognition

### Backend Infrastructure / 后端基础设施

- **User Vault Service**: Manages user credits and balances
- **Badge Sync Service**: Synchronizes on-chain badges with database
- **Event Indexer**: Tracks blockchain events in real-time
- **AI Integration**: Chatbot and digital identity services

### Frontend Experience / 前端体验

- **Wallet Authentication**: Seamless Web3 login
- **Quest Interface**: User-friendly task management
- **Badge Gallery**: Visual achievement showcase
- **Profile Dashboard**: Comprehensive user overview
- **AI Chat Interface**: Interactive digital identity
