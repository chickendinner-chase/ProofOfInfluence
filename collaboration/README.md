# AI 协作速记

> 一份简洁的 AI 协作参考，帮助团队在需要时快速对齐流程。

## 核心思路
- 明确目标：先写下想实现的 Link Bio 功能或问题。
- 规划再执行：让 AI（如 ChatGPT 或 Cursor）给出步骤，再着手编码。
- 快速迭代：实现后立刻自测，必要时再寻求 AI 协助优化。

## 基本流程
1. **整理需求**：记录要构建或修改的模块（前端、后端或样式）。
2. **生成方案**：在 AI 工具中描述需求，获取实现建议和注意事项。
3. **动手实现**：按照方案在本仓库中更新代码并运行测试命令。
4. **自测与复盘**：验证功能、总结经验，并在下一个任务前更新待办。

## 推荐工具
- **ChatGPT / Cursor**：用于方案讨论、代码补全和重构建议。
- **本地开发环境**：运行 `npm run dev` 或相关脚本验证效果。
- **版本控制**：通过 Git 记录每次改动，保持提交信息清晰。

需要更深入的配合时，可在此文件补充自己的团队约定。保持轻量，专注于交付 Link Bio 核心体验。
